## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/Gabrielhidro/animais_fantasticos

# Access
$ cd animais_fantasticos

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
